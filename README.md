# Windows-Fullscreen-Carousel
一款用于windows系统下的轮播工具，支持图片，url，本地html页面，视频轮播

# 特点
- 支持图片轮播，本地网页，网址加载
- 退出密码保护，防止误操作退出
- 支持多种方式轮播切换
- 支持双屏，一个屏幕设置，一个屏幕轮播
- 支持自定义显示文字
- 支持视频轮播
